# ACUCyS CTF

The Australian Council of University Cyber Societies (ACUCyS) coordinates and supports Capture The Flag (CTF) activity across our member university cyber clubs. This space collects write-ups, challenge walkthroughs, and shared learning from events run by ACUCyS and its member societies.

CTFs are a practical way for students to build skills in areas like web security, cryptography, reverse engineering, forensics, OSINT, and more. - Learn how different teams approached the same problem.
- Discover techniques and tools you can apply in future ACUCyS and club CTFs.
